## Conventions

- One spec per feature or verifier.
- Specs are numbered in the order they were proposed, not the order they ship.
- Cross-linked to the sibling [canon](https://github.com/Orthogon-AI-Labs/canon) specs when the work is part of a shared convention.
- Status values: `Draft`, `Ready for implementation`, `In progress`, `Shipped`.
